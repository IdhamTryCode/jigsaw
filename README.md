# 🧩 Jigsaw Puzzle Anak-anak

Sebuah game jigsaw puzzle interaktif berbasis web yang dirancang khusus untuk anak-anak usia 4-8 tahun.

## 📋 Fitur

- Menu utama dengan preview 9 gambar puzzle berbeda
- Puzzle yang dibagi menjadi potongan 2x2 atau 3x3
- Antarmuka drag-and-drop yang mudah digunakan
- Kotak puzzle yang luas (400x400 pixel) untuk memudahkan anak-anak bermain
- Efek magnet yang otomatis menarik potongan ke posisi yang benar ketika hampir tepat
- Getaran ringan saat magnet aktif (pada perangkat yang mendukung)
- Desain ramah anak dengan warna-warna cerah
- Efek suara untuk memberikan umpan balik positif
- Responsif untuk desktop dan perangkat mobile
- Animasi yang menyenangkan saat puzzle selesai

## 🚀 Cara Menjalankan

1. Clone repository ini ke komputer Anda
2. Buka file `index.html` di browser web favorit Anda
3. Mulai bermain!

## 🎮 Cara Bermain

1. Pilih gambar puzzle yang ingin kamu mainkan dari menu utama
2. Gunakan mouse atau sentuhan untuk menarik dan melepas potongan puzzle
3. Potongan puzzle akan otomatis tertarik dan mengunci ke tempat yang benar ketika didekatkan
4. Tempatkan setiap potongan di posisi yang benar untuk menyelesaikan puzzle
5. Gunakan dropdown untuk memilih tingkat kesulitan (2x2 atau 3x3)
6. Tekan tombol "Acak Ulang" untuk memulai puzzle baru
7. Tekan tombol "Kembali" untuk kembali ke menu utama

## 🖼️ Gambar Puzzle

Proyek ini menyertakan 9 gambar puzzle yang berbeda:
- `puzzle1.jpg` sampai `puzzle9.jpg`

Anda dapat mengganti gambar-gambar ini dengan gambar Anda sendiri di folder `public/images/`.
Ukuran ideal untuk gambar puzzle adalah 400x400 piksel.

## 🎵 Efek Suara

Proyek ini menggunakan dua efek suara:
- `correct.mp3`: Diputar saat potongan puzzle ditempatkan dengan benar
- `success.mp3`: Diputar saat seluruh puzzle selesai

Anda dapat mengganti file suara dengan file MP3 Anda sendiri di folder `public/sounds/`.

## 📝 Catatan

Proyek ini menggunakan HTML, CSS, dan JavaScript murni tanpa framework tambahan, sehingga sangat ringan dan mudah dijalankan di semua browser modern. Efek magnet dan animasi membuat permainan lebih menyenangkan dan membantu anak-anak untuk menyelesaikan puzzle dengan lebih mudah.
